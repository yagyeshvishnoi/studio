# **App Name**: TrueSight AI

## Core Features:

- User Authentication: Secure user login and signup using email and password with session-based or JWT authentication.
- Image Upload: Enable users to upload images (JPEG, PNG) via drag-and-drop or click-to-upload.
- AI Detection API Call: Send the uploaded image to the backend REST API for AI authenticity analysis.
- Result Display: Display the AI probability (float value from 0.0 to 1.0) both numerically (e.g., 55% AI) and graphically (animated gauge or pie chart).
- Graphical Visualization: Implement an interactive chart to display the probability using libraries like Recharts or Framer Motion.
- Dynamic Feedback Tool: Employ an AI tool to offer contextual text feedback (e.g., "Likely AI-generated" or "Likely real") based on the confidence score.

## Style Guidelines:

- Background color: Dark gray (#222222) to enhance the professional look.
- Primary color: Vibrant purple (#A78BFA) to convey trustworthiness and innovation.
- Accent color: Cyan (#7DD3FC) for interactive elements and highlights, creating contrast.
- Body and headline font: 'Inter' sans-serif for a modern, neutral feel.
- Use crisp, minimalist icons for a clean and modern interface.
- Implement a responsive design that works seamlessly across desktop and mobile devices.
- Incorporate animated transitions and micro-interactions using Framer Motion to create an engaging user experience.
